## 🎨 Design & Experiência do Usuário
O projeto foi redesenhado para oferecer uma experiência clara e objetiva:
- **Estética Clean:** Uso estratégico de espaços em branco e sombras suaves.
- **Tipografia Moderna:** Implementação da fonte *Plus Jakarta Sans* para legibilidade superior.
- **Interatividade:** Animações de entrada (AOS) e efeitos de hover nos cards de habilidades.
- **Micro-interações:** Header dinâmico que altera sua elevação ao rolar a página.

## 🛠️ Tecnologias e Bibliotecas
- **Linguagens:** HTML5, CSS3 (Variáveis e Flexbox/Grid), JavaScript (ES6+).
- **Animações:** [AOS.js](https://michalsnik.github.io/aos/) (Animate On Scroll).
- **Ícones:** [Font Awesome 6](https://fontawesome.com/).
- **Formulário:** Integração com [Formspree](https://formspree.io/) para recebimento de leads por e-mail.

## 📱 Responsividade
